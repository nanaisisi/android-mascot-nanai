デスクトップ固有機能

Windowsデスクトップ環境の特性を最大限活用した高度な機能群

👻
透過・フローティング表示
デスクトップ上に透過ウィンドウとしてキャラクターを表示。他のアプリケーションの邪魔にならない。
📌
常に最前面表示
すべてのアプリケーションの最前面に表示。重要な通知や常時表示したい情報に最適。
🖱️
ドラッグ移動・配置
キャラクターをドラッグしてデスクトップ上の任意の位置に配置。位置は自動保存される。
🔔
システムトレイ統合
システムトレイに常駐。右クリックメニューから各種機能にアクセス可能。
⌨️
グローバルホットキー
システム全体でのキーボードショートカット。どのアプリ使用中でも機能を呼び出し可能。
🖥️
デスクトップ環境連携
壁紙変更検出、ウィンドウフォーカス監視、マルチモニター対応など。
